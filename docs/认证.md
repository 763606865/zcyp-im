# 认证

## 1. 获取访问令牌

- 方法：`POST`
- 路径：`/im/auth/token`
- 鉴权：无需 Bearer Token

### 请求体

```json
{
  "app_code": "app_xxxxxxxx",
  "app_key": "your_app_key",
  "user_id": "u_1001"
}
```

### 说明

- `app_code` 必须存在
- `app_key` 必须匹配应用
- `user_id` 必须已经在该应用下创建，且用户状态为 `active`

### 成功响应

```json
{
  "code": 200,
  "data": {
    "app_code": "app_xxxxxxxx",
    "user_id": "u_1001",
    "token": "jwt_token",
    "expires_at": "2026-07-09T16:00:00+08:00"
  },
  "message": "",
  "meta": {
    "timestamp": "2026-07-09T15:00:00+08:00",
    "response_time": 3
  }
}
```

### 常见错误

```json
{
  "code": 401,
  "data": null,
  "message": "invalid app credentials",
  "meta": {
    "timestamp": "2026-07-09T15:00:00+08:00",
    "response_time": 2
  }
}
```

```json
{
  "code": 404,
  "data": null,
  "message": "user not found",
  "meta": {
    "timestamp": "2026-07-09T15:00:00+08:00",
    "response_time": 2
  }
}
```

```json
{
  "code": 403,
  "data": null,
  "message": "user disabled",
  "meta": {
    "timestamp": "2026-07-09T15:00:00+08:00",
    "response_time": 2
  }
}
```

## 2. Bearer Token 使用方式

HTTP 请求头：

```http
Authorization: Bearer <token>
```

Token 载荷包含：

- `app_code`
- `user_id`
- `iss`
- `iat`
- `exp`

## 3. WebSocket 连接鉴权

使用查询参数传递：

```text
GET ws://<ws-host>:<ws-port>/im/connect?token=<token>
```

如果 token 无效、用户不存在、用户已禁用，则连接会失败。
