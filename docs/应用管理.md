# 应用管理

## 1. 获取应用列表

- 方法：`GET`
- 路径：`/admin/apps`

### 成功响应

```json
{
  "items": [
    {
      "id": 1,
      "app_code": "app_xxxxxxxx",
      "name": "demo-app",
      "app_key": "app_key_value",
      "app_secret": "app_secret_value",
      "status": "active",
      "scenario": ["group", "live_room"],
      "created_at": "2026-07-08T15:00:00+08:00",
      "modified_at": "2026-07-08T15:00:00+08:00"
    }
  ]
}
```

## 2. 创建应用

- 方法：`POST`
- 路径：`/admin/apps`

### 请求体

```json
{
  "name": "demo-app",
  "scenario": ["single", "group", "chatroom", "live_room"]
}
```

### 成功响应

```json
{
  "id": 1,
  "app_code": "app_xxxxxxxx",
  "name": "demo-app",
  "app_key": "generated_app_key",
  "app_secret": "generated_app_secret",
  "status": "active",
  "scenario": ["single", "group"],
  "created_at": "2026-07-08T15:00:00+08:00",
  "modified_at": "2026-07-08T15:00:00+08:00"
}
```

## 3. 获取应用详情

- 方法：`GET`
- 路径：`/admin/apps/:app_code`

### 路径参数

- `app_code`：应用编码

### 常见错误

```json
{
  "error": "app not found"
}
```
