# 用户管理

## 1. 创建或更新用户

- 方法：`POST`
- 路径：`/admin/apps/:app_code/users`

### 请求体

```json
{
  "external_user_id": "u_1001",
  "nickname": "张三",
  "avatar_url": "https://example.com/avatar.png"
}
```

### 说明

- 同一个 `external_user_id` 已存在时，会更新昵称和头像
- 已存在用户的状态不会因为更新资料被自动重置

### 成功响应

```json
{
  "id": 1,
  "app_id": 1,
  "external_user_id": "u_1001",
  "nickname": "张三",
  "avatar_url": "https://example.com/avatar.png",
  "status": "active",
  "created_at": "2026-07-08T15:00:00+08:00",
  "updated_at": "2026-07-08T15:10:00+08:00"
}
```

## 2. 获取用户列表

- 方法：`GET`
- 路径：`/admin/apps/:app_code/users`

### 查询参数

- `limit`：可选，默认 `50`，范围 `1-200`

## 3. 获取单个用户

- 方法：`GET`
- 路径：`/admin/apps/:app_code/users/:external_user_id`

## 4. 更新用户状态

- 方法：`PATCH`
- 路径：`/admin/apps/:app_code/users/:external_user_id/status`

### 请求体

```json
{
  "status": "disabled"
}
```

### 支持状态

- `active`
- `disabled`

### 说明

- `disabled` 用户不能获取 token
- `disabled` 用户不能建立 WebSocket 连接
- `disabled` 用户不能发送消息

### 常见错误

```json
{
  "error": "user status invalid"
}
```
