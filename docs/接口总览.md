# 接口总览

本文档描述 `zcyp-im` 当前对外提供的 HTTP 和 WebSocket 接口。

## 基础信息

- 服务健康检查：`GET /health`
- API 服务前缀：`/admin`、`/im`
- WebSocket 网关路径：`/im/connect`
- 数据格式：`application/json`

## 鉴权规则

接口分两类：

1. 无需 Bearer Token

- `POST /im/auth/token`
- `/admin/*`

2. 需要 Bearer Token

- 除上面两项之外的 `/im/*` 接口

HTTP 认证头格式：

```http
Authorization: Bearer <token>
```

WebSocket 连接格式：

```text
GET ws://<ws-host>:<ws-port>/im/connect?token=<token>
```
